# PR.IP Test Procedures

## PR.IP-01 – Baseline Configuration Established

### Test Procedure
1. Review system documentation to verify that baseline configurations are defined for all critical systems.
2. Inspect configuration management tools or system images to confirm baseline configurations exist.
3. Verify that baseline configurations are documented and accessible to administrators.

### Expected Result
Baseline configurations are documented and consistently applied to systems across the environment.

---

## PR.IP-02 – Secure Configuration Management

### Test Procedure
1. Review configuration management policies and procedures.
2. Verify that systems are configured according to secure configuration standards.
3. Inspect sample systems to confirm configurations align with documented baselines.

### Expected Result
Systems follow documented secure configuration standards and deviations are controlled.

---

## PR.IP-03 – Configuration Change Control Processes

### Test Procedure
1. Review change management procedures related to configuration changes.
2. Examine change requests and approvals for recent configuration modifications.
3. Verify that configuration changes are documented, reviewed, and approved before implementation.

### Expected Result
Configuration changes follow formal change management processes with documented approvals.

---

## PR.IP-04 – Backups of Information Conducted

### Test Procedure
1. Review backup policies and procedures.
2. Verify backup schedules for critical systems and data.
3. Confirm that backup restoration procedures are tested periodically.

### Expected Result
Backups are performed regularly and restoration capabilities are validated.
