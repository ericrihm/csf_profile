# ID.RA-06: Risk Response Management

**Subcategory:** ID.RA-06 — Risk responses are chosen, prioritized, planned, tracked, and communicated
**NIST SP 800-53 Ref:** PM-09,PM-18,PM-30,RA-07

## Implementation Example

> Ex3: Track the progress of risk response implementation (e.g., plan of action and milestones [POA&M]

## Alma Security Implementation

Alma Security documents risk responses (accept, mitigate, transfer, avoid) in the risk register. Vulnerability management plan criteria guide remediation versus acceptance decisions. POA&Ms track risk response progress. Stakeholder communication of risk decisions occurs through governance meetings.
