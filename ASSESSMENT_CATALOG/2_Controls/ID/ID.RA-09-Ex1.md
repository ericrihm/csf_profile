## Alma Security Implementation

Alma Security evaluates software and hardware security posture during procurement through vendor security questionnaires. AWS Marketplace and approved vendor lists guide acquisition decisions. Software integrity verification (checksums, signatures) is performed for critical deployments but not standardized across all acquisitions.
