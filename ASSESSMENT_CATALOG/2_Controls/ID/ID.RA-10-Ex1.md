## Alma Security Implementation

Alma Security conducts supplier risk assessments for critical vendors using security questionnaires and SOC 2 report reviews. Third-party risk management policy defines assessment criteria. Critical supplier assessments are performed prior to contract signing. Ongoing monitoring of critical suppliers is annual.
