**Subcategory:** ID.RA-08 — Processes for receiving, analyzing, and responding to vulnerability disclosures are established

## Alma Security Implementation

Alma Security has a security contact email for receiving vulnerability disclosures. Received disclosures are triaged by the security team. No formal vulnerability disclosure program (VDP) or security.txt file is published. Supplier vulnerability information sharing is ad hoc.
