# ID.RA-03: Threat Identification and Recording

**Subcategory:** ID.RA-03 — Internal and external threats to the organization are identified and recorded
**NIST SP 800-53 Ref:** PM-12,PM-16,RA-03,SI-05

## Implementation Example

> Ex3: Implement processes for identifying internal threat actors

## Alma Security Implementation

Alma Security maintains a threat register documenting known external and internal threat actors relevant to the organization's industry and profile. Threat intelligence informs the register updates quarterly. Insider threat awareness is addressed through security training but no formal insider threat program exists.
