# ID.AM-04: Supplier Service Inventory

**Subcategory:** ID.AM-04 — Inventories of services provided by suppliers are maintained
**NIST SP 800-53 Ref:** AC-20,SA-09,SR-02

## Implementation Example

> Ex1: Inventory all external services used by the organization, including third-party infrastructure-

## Alma Security Implementation

Alma Security maintains a supplier service inventory in ServiceNow tracking SaaS, IaaS, and PaaS providers. New vendor onboarding requires IT and security review before provisioning. Contract management tracks service agreements and renewal dates.
