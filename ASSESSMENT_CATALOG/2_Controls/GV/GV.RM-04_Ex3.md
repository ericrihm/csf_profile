# GV.RM-04 Ex3: Document Shared Responsibility Conditions

**Subcategory:** GV.RM-04 — Strategic direction that describes appropriate risk response options is established and communicated
**NIST SP 800-53 Ref:** PM-09, PM-28, PM-30, SR-02

## Implementation Example

> Document conditions under which shared responsibility models are acceptable (e.g., outsourcing a function, using a cloud-based service).

## Alma Security Implementation

Alma Security documents shared responsibility conditions for all cloud services (AWS, SaaS tools) in the Third Party Risk Management Policy. Each critical vendor relationship includes a shared responsibility matrix reviewed during onboarding and annually. AWS shared responsibility is explicitly mapped to Alma's control obligations, and the CISO validates that internal controls cover Alma's portion of the shared model during quarterly reviews.
