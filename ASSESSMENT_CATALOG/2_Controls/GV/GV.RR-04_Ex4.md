# GV.RR-04 Ex4: Define Personnel Security Policy Obligations

**Subcategory:** GV.RR-04 — Cybersecurity is included in human resources practices
**NIST SP 800-53 Ref:** PM-13, PS-01, PS-07, PS-09

## Implementation Example

> Define and enforce obligations for personnel to be aware of, adhere to, and uphold security policies.

## Alma Security Implementation

All employees sign an acknowledgment of security policy obligations upon hire and annually. The Acceptable Use Policy defines expectations for system usage, data handling, and incident reporting. Non-compliance triggers a progressive response: first occurrence results in additional training, repeated violations escalate through HR disciplinary processes. Awareness is reinforced through quarterly phishing simulations and annual security training.
