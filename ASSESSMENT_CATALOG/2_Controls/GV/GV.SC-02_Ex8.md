# GV.SC-02 Ex8: Establish Information Sharing Rules

**Subcategory:** GV.SC-02 — Cybersecurity roles and responsibilities for suppliers, customers, and partners are established, communicated, understood, and enforced
**NIST SP 800-53 Ref:** SR-02, SR-03, SR-05

## Implementation Example

> Establish rules and protocols for information sharing and reporting processes between the organization and its suppliers.

## Alma Security Implementation

Alma Security defines information sharing protocols in vendor contracts: security incident notifications within 72 hours, quarterly compliance status reports from Tier 1 vendors, and annual SOC 2 report submissions. Information sharing classifications (TLP protocol) govern what threat intelligence is shared with which vendor tiers. All shared information is tracked in ServiceNow with audit trails and access controls.
