# GV.SC-02 Ex6: Develop Supplier Cybersecurity Roles

**Subcategory:** GV.SC-02 — Cybersecurity roles and responsibilities for suppliers, customers, and partners are established, communicated, understood, and enforced
**NIST SP 800-53 Ref:** SR-02, SR-03, SR-05

## Implementation Example

> Develop roles and responsibilities for suppliers, customers, and business partners to address shared cybersecurity risks.

## Alma Security Implementation

Alma Security defines cybersecurity roles for suppliers in contractual agreements: suppliers must designate a security contact, respond to security questionnaires within agreed SLAs, notify Alma of security incidents within 72 hours, and cooperate in joint incident response. Customer responsibilities for platform security (credential management, access controls) are documented in the customer security guide. Partner integration security requirements are defined in API partnership agreements.
