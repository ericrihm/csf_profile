# GV.RR-04 Ex2: Cybersecurity Knowledge in Hiring and Training

**Subcategory:** GV.RR-04 — Cybersecurity is included in human resources practices
**NIST SP 800-53 Ref:** PM-13, PS-01, PS-07, PS-09

## Implementation Example

> Consider cybersecurity knowledge to be a positive factor in hiring, training, and retention decisions.

## Alma Security Implementation

Alma Security includes cybersecurity competency as a valued skill in hiring decisions across engineering and IT roles. The company funds security certifications (CISSP, CCSP, Security+) and provides annual training budgets for the security team. Retention strategies include competitive compensation for security roles, career development paths, and conference attendance support. Security awareness is factored into performance reviews for all technical staff.
