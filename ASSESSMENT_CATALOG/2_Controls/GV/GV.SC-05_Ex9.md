# GV.SC-05 Ex9: Require Supplier Security Practice Evidence

**Subcategory:** GV.SC-05 — Requirements to address cybersecurity risks in supply chains are established, prioritized, and integrated into contracts and other agreements with suppliers and other relevant third parties
**NIST SP 800-53 Ref:** SA-04, SA-09, SR-03, SR-05, SR-06, SR-10

## Implementation Example

> Contractually require suppliers to provide evidence of performing acceptable security practices through, for example, independent assessments and certification.

## Alma Security Implementation

Vendor contracts require evidence of security practices tiered by criticality: Tier 1 vendors must provide annual SOC 2 Type II reports and penetration test summaries; Tier 2 vendors must provide completed security questionnaires and certifications (ISO 27001, SOC 2); Tier 3 vendors must attest to baseline security practices. Evidence is collected and tracked in ServiceNow with automated reminders for expiring certifications.
