# DE.CM-03 Ex3: Deception Technology Monitoring

**Subcategory:** DE.CM-03 — Personnel activity and technology usage are monitored to find potentially adverse events

**NIST SP 800-53 Ref:** AC-02, AU-12, AU-13, CA-07, CM-10, CM-11

## Implementation Example

> Continuously monitor deception technology, including user accounts, for any usage

## Alma Security Implementation

Alma Security has not deployed dedicated deception technology such as honeypots, honeytokens, or canary accounts. The capability is recognized as a detection enhancement opportunity but is not currently prioritized given the focus on establishing 24/7 monitoring coverage and completing the cross-source correlation scripts. Future consideration for deception technology will be evaluated after the Incident Response Enhancement project delivers its core objectives.
