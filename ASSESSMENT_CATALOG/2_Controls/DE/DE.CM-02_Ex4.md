# DE.CM-02 Ex4: Environmental Surveillance Systems

**Subcategory:** DE.CM-02 — The physical environment is monitored to find potentially adverse events

**NIST SP 800-53 Ref:** CA-07, PE-03, PE-06, PE-20

## Implementation Example

> Monitor the physical environment using alarm systems, cameras, and security guards

## Alma Security Implementation

The Redwood City office is monitored by a commercial security alarm system covering entry points, server room, and common areas with 24/7 central station monitoring. Security cameras cover building entrances, parking areas, and server room access corridors with 90-day retention on the recording system. Building security personnel are provided by the property management company during business hours, with alarm response contracted to a local security services provider for after-hours coverage.
