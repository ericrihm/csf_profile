# DE.CM-09 Ex5: Endpoint Cyber Health Monitoring

**Subcategory:** DE.CM-09 — Computing hardware and software, runtime environments, and their data are monitored to find potentially adverse events

**NIST SP 800-53 Ref:** AC-04, AC-09, AU-12, CA-07, CM-03, CM-06, CM-10, CM-11, SC-34, SC-35, SI-04, SI-07

## Implementation Example

> Use technologies with a presence on endpoints to detect cyber health issues (e.g., missing patches, malware infections)

## Alma Security Implementation

SentinelOne provides continuous endpoint health monitoring across all managed devices, detecting malware infections, unauthorized processes, and behavioral indicators of compromise in real time. The SentinelOne management console reports endpoint patch status, agent health, and protection policy compliance. Vulnerability scan summaries identify endpoints with missing critical patches, and findings are tracked through the remediation ticketing workflow managed by Nadia Khan's team.
