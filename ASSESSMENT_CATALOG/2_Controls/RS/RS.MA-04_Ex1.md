# RS.MA-04 Ex1: Ongoing Incident Status Tracking

**Subcategory:** RS.MA-04 — Incidents are escalated or elevated as needed

**NIST SP 800-53 Ref:** IR-04, IR-05, IR-06, IR-07

## Implementation Example

> Track and validate the status of all ongoing incidents

## Alma Security Implementation

Active incidents are tracked in ServiceNow with mandatory status updates at defined intervals based on severity level. The SOC Manager maintains visibility across all open incidents through a ServiceNow dashboard showing current status, assigned lead, severity, and time elapsed. Status validation occurs through direct communication between the incident lead and SOC Manager, confirming that documented status reflects actual response progress and scope understanding.
