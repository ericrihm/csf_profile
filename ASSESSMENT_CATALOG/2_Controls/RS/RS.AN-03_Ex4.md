# RS.AN-03 Ex4: Cyber Deception Technology for Attacker Behavior Analysis

**Subcategory:** RS.AN-03 — Analysis is performed to establish what has taken place during an incident and the root cause of the incident

**NIST SP 800-53 Ref:** AU-07, IR-04

## Implementation Example

> Check any cyber deception technology for additional information on attacker behavior

## Alma Security Implementation

Alma Security does not currently deploy dedicated cyber deception technology (honeypots, honeytokens, or deception platforms). Attacker behavior analysis relies on SentinelOne endpoint telemetry and GuardDuty findings rather than deception-based intelligence. This capability gap is acknowledged but not prioritized given the current focus on foundational incident response improvements through the IR Enhancement project.
