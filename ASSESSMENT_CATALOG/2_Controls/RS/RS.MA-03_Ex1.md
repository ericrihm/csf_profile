# RS.MA-03 Ex1: Incident Type Categorization

**Subcategory:** RS.MA-03 — Incidents are categorized and prioritized

**NIST SP 800-53 Ref:** IR-04, IR-05, IR-06

## Implementation Example

> Further review and categorize incidents based on the type of incident (e.g., data breach, ransomware, DDoS, improper usage)

## Alma Security Implementation

SOC analysts categorize validated incidents using a documented taxonomy that includes phishing, malware/ransomware, unauthorized access, data exfiltration, DDoS, insider threat, and misconfiguration categories. Categorization is recorded in ServiceNow SOC tickets and drives playbook selection for type-specific response procedures. The D&R team maintains categorization data for trending analysis and annual incident reporting to leadership.
