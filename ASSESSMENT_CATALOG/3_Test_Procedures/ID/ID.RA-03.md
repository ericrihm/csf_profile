# ID.RA-03: Threat Identification and Recording Test Procedures

**CSF Subcategory:** ID.RA-03 - Internal and external threats to the organization are identified and recorded

---

## Test Procedures

1. **Review Threat Register**
   - Obtain the current threat register
   - Verify external threat actors are profiled by industry relevance
   - Confirm internal threat categories are documented
   - Check that threat register is updated quarterly

2. **Validate External Threat Identification**
   - Review threat actor profiles for completeness (TTPs, targets)
   - Verify threat intelligence sources inform threat identification
   - Check for industry-specific threat assessments
   - Confirm threat register reflects current threat landscape

3. **Assess Internal Threat Identification**
   - Review insider threat awareness training content
   - Check for formal insider threat program documentation
   - Verify behavioral detection capabilities (SentinelOne)
   - Assess processes for identifying internal threat indicators

4. **Review Threat Hunting Activities**
   - Obtain threat hunting activity logs or reports
   - Verify threat hunting cadence and coverage
   - Check for threat hunting playbooks or procedures
   - Review findings from recent threat hunting activities

5. **Validate Threat-to-Asset Mapping**
   - Check if threat actors are mapped to targeted assets
   - Verify threats are mapped to exploitable vulnerabilities
   - Confirm threat register informs risk assessment updates
   - Review how threat information drives defensive priorities

## Evidence Requests

- [ ] Threat register (current version)
- [ ] Threat actor profiles
- [ ] Insider threat awareness training materials
- [ ] Threat hunting activity reports
- [ ] Quarterly threat register update records
- [ ] Threat-to-asset mapping documentation

## Notes

The absence of a formal insider threat program is a notable gap. Assess whether existing behavioral detection tools provide adequate compensating capability for internal threat identification.
