# ID.RA-09: Acquisition Integrity Assessment Test Procedures

**CSF Subcategory:** ID.RA-09 - The authenticity and integrity of hardware and software are assessed prior to acquisition and use

---

## Test Procedures

1. **Review Procurement Security Process**
   - Obtain vendor security questionnaire template
   - Verify procurement process includes security evaluation
   - Review approved vendor list and selection criteria
   - Check for supply chain integrity requirements

2. **Validate Software Integrity Verification**
   - Review software integrity verification procedures
   - Check for checksum or signature verification for critical software
   - Assess SBOM requirements for procured software
   - Verify code signing policies for internal development

3. **Assess Hardware Authenticity**
   - Review hardware procurement from authorized distributors
   - Check for hardware authenticity verification procedures
   - Verify counterfeit detection capabilities
   - Assess supply chain security for hardware components

4. **Review Acquisition Risk Assessment**
   - Sample 3 recent technology acquisitions
   - Verify security assessment was completed prior to acquisition
   - Confirm risk findings were addressed before deployment
   - Check for post-acquisition security validation

5. **Evaluate Ongoing Integrity Monitoring**
   - Verify software update integrity verification
   - Check for file integrity monitoring on critical systems
   - Assess trusted source policies for software downloads
   - Review AWS Marketplace usage and approval process

## Evidence Requests

- [ ] Vendor security questionnaire template
- [ ] Approved vendor list
- [ ] Software integrity verification records
- [ ] Recent acquisition security assessments (3)
- [ ] Hardware procurement authorization records
- [ ] Code signing policy documentation

## Notes

Software supply chain integrity is increasingly critical. Assess whether the organization can verify the provenance of software components through SBOMs and signature verification at scale.
