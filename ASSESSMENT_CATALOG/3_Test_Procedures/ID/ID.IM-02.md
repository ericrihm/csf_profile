# ID.IM-02: Security Test-Based Improvement Test Procedures

**CSF Subcategory:** ID.IM-02 - Improvements are identified from security tests and exercises, including those done in coordination with suppliers

---

## Test Procedures

1. **Review Security Testing Program**
   - Obtain security testing program documentation
   - Verify phishing simulation schedule and scope
   - Review tabletop exercise documentation
   - Check for penetration testing plans or reports

2. **Validate Improvement Identification from Tests**
   - Review phishing simulation results and identified improvements
   - Check tabletop exercise after-action reports
   - Verify improvements are documented and tracked
   - Confirm findings inform training and control updates

3. **Assess Exercise Scope and Participation**
   - Verify internal stakeholder involvement in exercises
   - Check for supplier coordination in exercises
   - Review exercise scenario coverage and realism
   - Assess exercise frequency against industry benchmarks

4. **Review Penetration Testing**
   - Check for penetration testing history or plans
   - Verify penetration testing scope and methodology
   - Review any red team or purple team exercise results
   - Assess plans for maturing offensive security testing

5. **Evaluate Improvement Tracking**
   - Verify improvement items from exercises are tracked
   - Check completion rates for identified improvements
   - Assess whether improvements are integrated into project management
   - Review metrics tracking from security tools and services

## Evidence Requests

- [ ] Security testing program documentation
- [ ] Phishing simulation results and metrics
- [ ] Tabletop exercise after-action reports
- [ ] Penetration testing reports (if performed)
- [ ] Improvement tracking from exercises
- [ ] Exercise participation records

## Notes

The absence of penetration testing is a significant gap for identifying technical vulnerabilities that scanning tools may miss. Assess plans for incorporating offensive security testing into the program.
