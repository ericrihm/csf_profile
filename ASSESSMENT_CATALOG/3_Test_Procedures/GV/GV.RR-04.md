# GV.RR-04: Human Resources Practices Test Procedures

**CSF Subcategory:** GV.RR-04 - Cybersecurity is included in human resources practices

---

## Test Procedures

1. **Review HR-security integration**
   - Request HR policies that include cybersecurity provisions
   - Verify cybersecurity is factored into hiring criteria for relevant roles
   - Confirm onboarding includes security awareness training and policy acknowledgment

2. **Examine background check practices**
   - Verify background checks are conducted for personnel in sensitive roles
   - Check for periodic re-screening requirements for privileged access holders
   - Confirm process covers both employees and contractors with system access

3. **Validate personnel security obligations**
   - Review employee handbook or policy for security behavior expectations
   - Verify personnel acknowledge security policies at hire and annually
   - Check that consequences for policy violations are documented

4. **Interview HR on cybersecurity integration**
   - Ask how cybersecurity knowledge factors into hiring and promotion decisions
   - Verify training and retention strategies consider security skill development
   - Confirm termination procedures include security-relevant steps (access revocation, asset return)

## Evidence Requests

- [ ] HR policies with cybersecurity provisions
- [ ] Background check policy and recent completion evidence
- [ ] Security policy acknowledgment records
- [ ] Termination checklist with security steps

## Notes

As a security company, Alma should demonstrate above-average integration of cybersecurity into HR practices. Assessment should verify this extends beyond the security team to all personnel with system access.
