# GV.SC-10: Supplier Termination Provisions Test Procedures

**CSF Subcategory:** GV.SC-10 - Cybersecurity supply chain risk management plans include provisions for activities that occur after the conclusion of a partnership or service agreement

---

## Test Procedures

1. **Review supplier termination procedures**
   - Request documented procedures for terminating supplier relationships
   - Verify procedures address both normal and adverse termination circumstances
   - Confirm data return and disposal requirements are defined

2. **Validate access deactivation practices**
   - Check process for deactivating supplier access upon relationship termination
   - Verify timeliness requirements for access revocation
   - Review recent termination examples for evidence of prompt access removal

3. **Examine end-of-life and obsolescence planning**
   - Request plans for component end-of-life maintenance and obsolescence
   - Verify critical dependencies have transition plans
   - Check that supplier termination risks are assessed and mitigated

4. **Interview Gerry on supplier offboarding process**
   - Ask about the supplier termination workflow in ServiceNow
   - Verify data leakage risks are addressed during termination
   - Confirm asset return or disposal verification processes

## Evidence Requests

- [ ] Supplier termination procedures
- [ ] Access revocation evidence from recent terminations
- [ ] End-of-life and obsolescence planning documentation
- [ ] Data disposal verification records

## Notes

Post-relationship risk management is often overlooked. Assessment should verify Alma has proactive procedures rather than ad hoc responses to supplier terminations.
