# GV.RR-03: Resource Allocation Test Procedures

**CSF Subcategory:** GV.RR-03 - Adequate resources are allocated commensurate with the cybersecurity risk strategy, roles, responsibilities, and policies

---

## Test Procedures

1. **Review security budget and staffing**
   - Request cybersecurity budget documentation and allocation breakdown
   - Verify staffing levels against documented requirements (15-person team vs. targets)
   - Confirm R-01 (understaffing risk) treatment plan includes budget for additional hires

2. **Validate resource-to-risk alignment**
   - Cross-reference resource allocation with risk treatment plans
   - Verify tools and technology investments support risk mitigation goals
   - Check that compensating controls (CrowdStrike MDR) are funded when staffing gaps exist

3. **Examine management review of resource adequacy**
   - Request evidence of periodic management reviews of resource sufficiency
   - Verify CISO raises resource gaps to CEO and Board
   - Check for documented resource requests and approval/denial decisions

4. **Interview CISO on resource constraints**
   - Ask whether current resources are adequate for the cybersecurity strategy
   - Verify process for requesting additional resources when needs change
   - Confirm resource allocation considers people, processes, and technology

## Evidence Requests

- [ ] Cybersecurity budget documentation
- [ ] Staffing plan and current vs. target headcount
- [ ] Resource request and approval records
- [ ] Compensating control funding documentation

## Notes

R-01 (understaffing at ~50% of target) is a documented High/High risk. Assessment should evaluate whether resource allocation decisions are responsive to this known gap and whether interim compensating controls are adequately funded.
